Selamat anda sudah 1 step yaitu mendownload extraordinary cbt rosetta.

Langkah-langkah installasi.

1. Postgresql & install
bisa ikuti tutor: https://www.digitalocean.com/community/tutorials/how-to-install-postgresql-on-ubuntu-20-04-quickstart

2. PGAdmin (optional)
bisa ikuti tutor: https://musaamin.web.id/cara-install-pgadmin4-ubuntu2204/

3. Restore exo-dump-master.sql bisa pakai PGAdmin atau pg-dump.

4. Edit .env, sesuaikan dengan konfigurasi database
  - biarkan SERVER_HOST dan SERVER_ASSET_URL kosong jika tidak melakukan konfigurasi advance
  - STORAGE_PATH isi dengan path folder storage pada direktori ini. contohnya C:\exo\storage

5. Jalankan main, masukkan secret key yang didapat dari https://ecosystem.ekstraordinary.com/
6. Untuk masuk halaman admin pakai /adm contoh http://host:port/adm
  - email: admin@shellrean.id
  - password: criticalpassword
7. Have nice day, jika mengalami kendala bisa tulis di https://github.com/shellrean-dev/exo-cbt-4-rosetta-issue/issues
